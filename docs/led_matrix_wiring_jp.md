# 5x4 LEDマトリクス配線

## 基本構成

20個のLEDを、5列 x 4行に接続します。

- 列側: LEDのアノード（A、長い足）
- 行側: LEDのカソード（K、短い足・本体の平らな側）
- 列はU2（74HC595）の出力から330Ωを通して駆動
- 行はNPNトランジスタでGNDへ落として選択
- 同時に選択する行は必ず1行だけ

```text
U2出力 -> 330Ω -> BUS/COL -> LEDアノード
LEDカソード -> ROW -> NPNコレクタ
NPNエミッタ -> GND
```

## 列側の接続

330Ωの独立型集合抵抗は、U2とBUSの間に入れます。

| U2出力 | U2ピン | 抵抗 | 接続先 |
|---|---:|---|---|
| QA | 15 | 330Ω | BUS0 / COL0 |
| QB | 1 | 330Ω | BUS1 / COL1 |
| QC | 2 | 330Ω | BUS2 / COL2 |
| QD | 3 | 330Ω | BUS3 / COL3 |
| QE | 4 | 330Ω | BUS4 / COL4 |

BUS0～BUS4は、同じ列にある4個のLEDのアノードへ接続します。

```text
BUS0 -> LED(0,0) A
     -> LED(0,1) A
     -> LED(0,2) A
     -> LED(0,3) A
```

BUS1～BUS4も同様です。

## 行側の接続

同じ行にある5個のLEDのカソードをまとめて、対応するNPNトランジスタの
コレクタへ接続します。

| 行 | LEDカソード | NPN |
|---|---|---|
| ROW0 | LED(0,0)～LED(4,0) | Q_ROW0 |
| ROW1 | LED(0,1)～LED(4,1) | Q_ROW1 |
| ROW2 | LED(0,2)～LED(4,2) | Q_ROW2 |
| ROW3 | LED(0,3)～LED(4,3) | Q_ROW3 |

各NPNの接続:

```text
U3出力 -> 4.7kΩ -> NPNベース
                       |
                     100kΩ
                       |
                      GND

NPNエミッタ -> GND
NPNコレクタ -> 対応するROW
```

U3との対応:

| U3出力 | U3ピン | 選択する行 |
|---|---:|---|
| QA | 15 | ROW0 |
| QB | 1 | ROW1 |
| QC | 2 | ROW2 |
| QD | 3 | ROW3 |

## 20個すべての接続

表内の各LEDは「アノード側の列」と「カソード側の行」の交点です。

| | COL0 / BUS0 | COL1 / BUS1 | COL2 / BUS2 | COL3 / BUS3 | COL4 / BUS4 |
|---|---|---|---|---|---|
| ROW0 | D00 | D10 | D20 | D30 | D40 |
| ROW1 | D01 | D11 | D21 | D31 | D41 |
| ROW2 | D02 | D12 | D22 | D32 | D42 |
| ROW3 | D03 | D13 | D23 | D33 | D43 |

例えばD21は次のように接続します。

```text
D21アノード   -> BUS2
D21カソード   -> ROW1
ROW1           -> Q_ROW1コレクタ
Q_ROW1エミッタ -> GND
```

## 1行分の配線例

ROW0は次の形です。

```text
BUS0 --|>|--+
BUS1 --|>|--+
BUS2 --|>|--+--- ROW0 --- Q_ROW0 collector
BUS3 --|>|--+              Q_ROW0 emitter --- GND
BUS4 --|>|--+
```

`|>|` の左側がアノード、右側がカソードです。

抵抗はLEDごとではなく、U2出力とBUSの間に1本ずつ置きます。

```text
U2 QA -> 330Ω -> BUS0
U2 QB -> 330Ω -> BUS1
U2 QC -> 330Ω -> BUS2
U2 QD -> 330Ω -> BUS3
U2 QE -> 330Ω -> BUS4
```

## 点灯例

D21（COL2、ROW1）だけを点灯する場合:

```text
U2 BUSデータ:
BUS4 BUS3 BUS2 BUS1 BUS0
  0    0    1    0    0

U3選択データ:
ROW3 ROW2 ROW1 ROW0
  0    0    1    0
```

U2のBUS2がHIGHになってLEDのアノードへ電流を供給し、Q_ROW1がONに
なってROW1をGNDへ落とすため、D21だけが点灯します。

ROW1でCOL0、COL2、COL4を同時に点灯する場合:

```text
U2 BUSデータ = 0b00010101
U3 ROW選択   = ROW1のみON
```

## 走査順序

表示更新は以下を繰り返します。

1. ROW0を選択し、ROW0の5ビットをBUSへ出す
2. ROW1を選択し、ROW1の5ビットをBUSへ出す
3. ROW2を選択し、ROW2の5ビットをBUSへ出す
4. ROW3を選択し、ROW3の5ビットをBUSへ出す
5. 7セグ桁0を表示
6. 7セグ桁1を表示

切り替える直前に、いったんU3の行/桁選択をすべてOFFにしてください。
これを行わないと、シフト中のデータで別のLEDが薄く点灯することがあります。

## 配線時の注意

- LEDの向きを20個すべて統一する
- NPNのE/B/C配列は品番ごとに確認する
- 330Ω集合抵抗はISOL（独立型）を使う
- 行選択トランジスタを同時に2個以上ONにしない
- U2、U3それぞれのVCC付近に0.1uFを置く
- ブレッドボード試験では、最初にROW0だけを固定して確認する
